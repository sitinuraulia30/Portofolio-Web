Projets

Project 1

Prototype Mobile

Proyek ini bertujuan untuk mendesain prototipe aplikasi mobile perpustakaan digital yang dirancang untuk mempermudah pengguna dalam mengakses koleksi buku, jurnal, dan sumber daya lainnya secara online. Desain prototipe ini didesain menggunakan Figma dan mencakup fitur utama seperti pencarian koleksi berbasis kategori, antarmuka peminjaman dan pengembalian buku secara digital, sistem notifikasi pengingat, serta integrasi dengan sistem manajemen perpustakaan.

Project 2

Aplikasi Web Perpustakaan Digital

Aplikasi Web Perpustakaan Digital ini dirancang untuk mempermudah pengelolaan dan akses koleksi perpustakaan secara online. Dibangun menggunakan HTML, PHP, dan MySQL, aplikasi ini memiliki fitur utama seperti pencarian koleksi, sistem peminjaman dan pengembalian buku secara digital, manajemen koleksi oleh admin, serta riwayat transaksi pengguna. Proyek ini berfokus pada antarmuka yang responsif, kemudahan penggunaan, dan efisiensi pengelolaan data perpustakaan.

Project 3

Desain Poster

Saya telah menyelesaikan proyek desain beberapa poster.
